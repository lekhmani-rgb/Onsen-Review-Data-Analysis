Onsen-Review-Data-Analysis

Overview
This project analyzes tourist reviews of onsen (hot springs) in Beppu, Japan.  
The goal is to understand customer sentiment and identify key themes in reviews.

What I Did
- Collected real-world review data manually
- Cleaned dataset (removed missing values and duplicates)
- Analyzed review sentiment (Positive, Negative, Mixed)
- Identified main themes (price, service, facilities, etc.)
- Created cross-tab analysis for deeper insights

Key Findings
- Most reviews are positive
- "Experience" and "Facilities" are the most common themes
- Water temperature and service also influence satisfaction

Tools Used
- Python
- Pandas
